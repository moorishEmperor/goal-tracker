# Goal Tracker - Personal Development Web App

Production-ready Flask application for tracking personal development goals.

## Features
- User authentication
- Goal creation with deadlines
- Task management with drag-and-drop
- Progress tracking
- Secure session management

## Tech Stack
- Flask 3.0
- PostgreSQL (production)
- SQLAlchemy ORM
- Gunicorn WSGI server
- Deployed on Render.com

## Local Development
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Access at: http://localhost:5000
